# 2026 국회 고위공직자 상장주식 대시보드

2026년 국회 고위공직자(국회의원·입법부 고위직) 재산정보 공개본의 **상장주식** 부문을 전수 집계한 인터랙티브 대시보드입니다.

🔗 **라이브:** https://leejeonghwan.github.io/assembly-stocks-2026/

## 주요 발견
- 보유자 수 1·2위가 **삼성전자(63명)·엔비디아(62명)**로 사실상 동률, 상위권을 미국 빅테크가 점령
- 지난 1년 매매 흐름: **한국 대형주는 순매도(삼성전자우·KB금융·대한항공·카카오뱅크), 미국 AI주는 순매수(엔비디아 +38·알파벳 +28·브로드컴·팔란티어)**
- 개인 보유액 1위 안철수 의원 본인(안랩) 약 1조 1,786억원 — 2위의 약 22배
- 양자컴퓨팅·우주·SMR 원전 등 테마주에도 다수 보유

## 데이터
- 출처: 2026년 국회 고위공직자 재산정보 공개본(상세)
